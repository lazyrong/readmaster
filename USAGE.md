## 📋 核心功能

### 一、脉络管理 (信息组织)

**什么是脉络?**
脉络是一个主题化的信息聚合单元，可以包含多个信息源和分析师。

**已有的示例脉络:**
1. 📈 **投资决策脉络** - 跟踪市场动态、财经新闻
2. 💼 **产品趋势脉络** - 关注产品创新、用户体验
3. 🎯 **竞品监控脉络** - 实时跟踪竞争对手动态

**如何创建新脉络?**
1. 点击左侧"我的脉络"右侧的 ➕ 按钮
2. 输入脉络名称（如："科技新闻聚合"）
3. 输入描述（可选）
4. 输入图标（可选，如：🚀）

---

### 二、添加信息源

**当前支持的信息源类型:**
- **RSS**: 任何支持 RSS/Atom 的网站
- **YouTube**: YouTube 频道或播放列表

**如何添加RSS信息源?**

1. 点击左侧"信息源"右侧的 ➕ 按钮
2. 输入信息源名称，如："36氪科技新闻"
3. 选择类型: `rss`
4. 输入 RSS URL，如：`https://36kr.com/feed`

**推荐的RSS源:**
```
科技媒体:
- 36氪: https://36kr.com/feed
- TechCrunch: https://techcrunch.com/feed
- Product Hunt: https://www.producthunt.com/feed
- Hacker News: https://news.ycombinator.com/rss

投资金融:
- 华尔街见闻: (需要查找具体RSS地址)
- 雪球: (需要查找具体RSS地址)
```

**如何添加YouTube频道?**

1. 找到YouTube频道ID
   - 访问频道页面
   - URL类似: `https://www.youtube.com/channel/UCxxxxxxxxx`
   - 复制 `UCxxxxxxxxx` 部分

2. 添加信息源
   - 名称: "李永乐老师"
   - 类型: `youtube`
   - URL: (输入频道主页URL)
   - 配置: `{"channelId": "UCxxxxxxxxx"}`

### 三、同步内容

**手动同步单个信息源:**
1. 找到信息源卡片
2. 点击右侧的 🔄 同步按钮
3. 等待同步完成提示

**批量同步所有信息源:**
1. 点击内容中心右上角的"同步"按钮
2. 系统会依次同步所有信息源
3. 同步完成后内容自动刷新

### 四、使用AI分析师

**内置分析师列表:**

1. **要点提炼师** ⭐️ (推荐首选)
   - 快速提取文章核心观点
   - 生成5条关键要点
   - 适用场景：快速浏览、会议纪要

2. **SWOT分析师** 💼
   - 商业战略分析
   - 优势、劣势、机会、威胁
   - 适用场景：项目评估、竞品分析

3. **竞品分析师** 🎯
   - 产品功能拆解
   - 创新点提取
   - 适用场景：产品调研、功能学习

4. **价值投资顾问** 💰
   - 投资机会评估
   - 财务健康度分析
   - 适用场景：投资决策、公司研究

5. **用户需求洞察师** 👥
   - 挖掘用户痛点
   - 行为洞察
   - 适用场景：需求分析、产品设计

**如何使用分析师分析内容?**

1. **选择分析师**
   - 点击右侧分析师卡片进行选中
   - 被选中的分析师会高亮显示

2. **分析内容**
   - 在内容卡片上点击"分析"按钮
   - 等待AI生成分析结果
   - 结果会显示在右侧面板

**注意事项:**
- ⚠️ 分析功能需要配置 OpenAI API Key
- 💡 建议先选择"要点提炼师"快速了解内容
- 📊 不同分析师可以对同一内容提供不同视角

### 五、内容过滤 (高级功能)

**为什么需要过滤?**
- 减少信息噪音
- 聚焦关注领域
- 提高阅读效率

**如何配置过滤规则?**

当前需要在创建/编辑信息源时设置JSON配置：

```json
{
  "url": "https://example.com/feed",
  "filter_rules": {
    "keywords": ["AI", "机器学习", "创业"],
    "exclude": ["广告", "推广"],
    "min_length": 100,
    "max_length": 5000
  }
}
```

**过滤规则说明:**
- `keywords`: 必须包含的关键词（任一匹配即可）
- `exclude`: 排除的关键词（包含任一即过滤）
- `min_length`: 最小内容长度
- `max_length`: 最大内容长度

## 💡 使用技巧

### 场景一：早晨信息浏览

1. 打开"投资决策脉络"
2. 点击"同步"获取最新内容
3. 使用"要点提炼师"快速浏览每条新闻
4. 对感兴趣的内容使用"价值投资顾问"深度分析

### 场景二：产品竞品调研

1. 创建专门的"竞品调研"脉络
2. 添加竞品公司的RSS源、公众号源
3. 设置关键词过滤："功能更新"、"新版本"
4. 使用"竞品分析师"和"SWOT分析师"组合分析

### 场景三：趋势研究

1. 创建"AI趋势"脉络
2. 添加TechCrunch、Hacker News等源
3. 设置关键词："GPT"、"Large Language Model"、"AI"
4. 使用"要点提炼师"整理趋势报告

## 🔧 常见问题

### Q1: 为什么内容为空?

**A:** 需要手动同步信息源
1. 点击信息源右侧的同步按钮
2. 或点击内容中心的批量同步

### Q2: 分析功能报错?

**A:** 需要配置OpenAI API Key
- 当前版本需要管理员配置环境变量 `OPENAI_API_KEY`
- 后续版本会支持用户自行配置

### Q3: 如何添加微信公众号?

**A:** 暂不支持
- 微信公众号需要特殊的抓取适配器
- 预计在 Phase 2 开发

### Q4: 能否导出分析结果?

**A:** 导出功能开发中
- 计划支持 Markdown 和 PDF 格式
- 预计在 Phase 2 完成

## 🚀 高级用法

### 自定义分析师

1. 点击右侧"AI分析师"标题旁的 ➕ 按钮
2. 输入分析师名称，如："技术可行性评估师"
3. 选择类别（框架/领域专家/数据洞察/内容处理）
4. 编写系统提示词（Prompt）:

```
你是一位资深技术架构师。请从技术可行性角度分析内容中提到的技术方案：
1. 技术难度评估
2. 所需技术栈
3. 潜在技术风险
4. 实现时间估算
5. 替代技术方案
```

5. 保存后即可使用
